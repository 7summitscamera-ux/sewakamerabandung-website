Workflow Wedding-Day

Workflow paling efisien — pickup H-1 sore, return H+1

Mayoritas wedding photographer Bandung pakai pola pickup H-1 sore (test gear di studio dulu) → akad pagi H + resepsi sore H → return H+1 pagi setelah loadout. Total cycle: 3 hari kalender = bayar 2 hari (promo 3-bayar-2). Hemat ~Rp 500-700rb per wedding.

  1. H-3 / H-2: Booking online + konfirmasi paket. Diskusi backup unit kalau perlu.
  2. H-1 sore: Pickup gear di Sriwijaya (untuk venue Setiabudi) atau Cisaranten (untuk venue Bandung Timur). Test setup di studio.
  3. Hari-H pagi: Akad shoot. Backup unit standby di cabang kalau dibutuhkan.
  4. Hari-H sore: Resepsi cover. Damage Waiver active untuk gear high-value.
  5. H+1 pagi: Return gear ke cabang asal. Total 3 hari = 2 hari bayar.

FAQ Wedding

Pertanyaan dari wedding photographer Bandung

Berapa harga sewa kamera untuk wedding di Bandung?

Paket Wedding Pro Setup Rp 1.5jt/hari (2× Sony A7IV + 24-70 GM + 70-200 GM + Godox V1 + softbox). Untuk cinema-grade tambah FX3 = Rp 2.2jt/hari. Promo 3 hari bayar 2 berlaku — total 3 hari Wedding Pro Rp 3jt (cover akad-resepsi-loadout).

Apakah ada backup unit untuk wedding day?

Ya — untuk wedding photographer pro dengan project nilai tinggi, kami sediakan backup unit Sony A7IV / FX3 on-call. Konfirmasi H-1 untuk arrange. Damage Waiver Protection (Rp 50rb/transaksi) sangat direkomendasikan, cover gear hingga Rp 5jt.

Bisa antar gear ke venue wedding di Bandung?

Bisa via ojek online ke seluruh venue Bandung. Untuk Padma Hotel, Sheraton, GH Universal area Setiabudi — antar dari Sriwijaya 7-10 menit. Mayoritas wedding photographer pickup gear H-1 sore lalu return H+1 setelah resepsi.

Apakah perlu bawa gear ke akad pagi 06:00?

Untuk akad pagi sebelum jam buka cabang, request antar khusus H-1 — kami bisa atur. Atau pickup H-1 sore biar gear sudah di tangan sebelum hari-H. Pickup H-1 sore + return H+1 pagi cocok dengan promo 3 hari bayar 2.

Setup ideal untuk wedding videografer cinematic?

Sony FX3 + Sony 24-70 GM + 70-200 GM + DJI RS3 Pro gimbal + Hollyland Lark M2 wireless mic + Saramonic XLR adapter + Aputure 100D lighting. Total ~Rp 2.2jt/hari. Untuk indie-film style, tambah lensa cinema Samyang Xeen.

Bagaimana sistem deposit untuk wedding pro?

KTP + KTP Penjamin + foto selfie pegang KTP + deposit (cash atau transfer). Untuk paket Wedding Pro nilai tinggi, deposit menyesuaikan. Untuk klien repeat (3+ booking), deposit bisa di-waive — chat admin untuk diskusi.
