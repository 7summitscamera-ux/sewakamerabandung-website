Pertanyaan dari indie filmmaker Bandung

Berapa harga sewa setup cinema-grade untuk indie film?

Paket Sineas Indie Rp 2.2jt/hari — Sony FX3 + lensa cinema Samyang Xeen 24+50 T1.5 + Aputure 300X LED + DJI RS4 Pro gimbal + Saramonic XLR audio + ND filter set. Promo 3 hari bayar 2 berlaku — total 3 hari Rp 4.4jt (cukup untuk 2-day shoot + buffer).

Apakah cocok untuk mahasiswa TA film Sineas Tel-U / Unpad / IKJ?

Sangat cocok. Banyak mahasiswa Tel-U Sineas, Fikom Unpad, dan IKJ yang sewa untuk TA film festival-aim. Tunjukkan KTM untuk priority service. Untuk shoot 5+ hari, ada special long-term rate. Untuk festival submission yang butuh setup pro, paket ini standar.

Cinema lens (Samyang Xeen) atau photo lens (Sony GM)?

Cinema lens (Xeen) untuk minimal focus breathing + manual focus precision (270° rotation) + T-stop accuracy. Photo lens (Sony GM) untuk autofocus + lebih ringan + budget lebih terjangkau. Untuk indie film tradisional, Xeen worth investment. Untuk run-and-gun documentary, Sony GM cukup. Detail di panduan lensa-cinema-vs-photo.

Sony FX3 atau FX30 untuk indie film?

FX3 full-frame, 4K 120p, dynamic range 15-stop — flagship cinema mirrorless. FX30 APS-C, 4K 120p, lebih affordable. Untuk festival submission yang butuh widest DR + low-light superior = FX3 (Rp 650rb/hari). Untuk indie film budget-friendly atau mahasiswa = FX30 (Rp 525rb/hari).

Apakah kami bantu konsultasi setup untuk script TA film?

Ya, free consultation via WhatsApp. Sebutkan: genre film, mood/tone (cinematic, documentary, horror, dll), shooting environment (indoor/outdoor/hybrid), budget. Tim akan rekomendasi setup paling cost-efficient untuk kebutuhan kamu.

Bisa sewa Tilta Nucleus follow focus untuk wireless focus pull?

Ya, Tilta Nucleus-M tersedia untuk sewa Rp 350rb/hari. Wireless follow focus untuk 3-motor (focus/iris/zoom). Cocok untuk dialog dynamic + AC pull focus dari jauh. Standar setup indie film professional.
