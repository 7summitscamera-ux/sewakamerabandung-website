FAQ Content Creator

Pertanyaan dari content creator Bandung

Setup gear paling cocok untuk vlog YouTube pemula?

Sony A7CII (compact 514g) + Sony 16-35mm f/2.8 GM (wide untuk vlog handheld) + Hollyland Lark M1 wireless mic. Total ~Rp 750rb/hari. Untuk gimbal smooth motion, tambah DJI RS3 Mini (Rp 200rb).

Berapa harga paket Vlogger Setup di 7summits Bandung?

Paket Vlogger Setup Rp 850rb/hari — Sony A7CII/FX30 + 16-35mm GM + DJI RS3 Mini gimbal + Hollyland Lark M2 wireless mic + 2 baterai cadangan. Promo 3 hari bayar 2 berlaku — total 3 hari Rp 1.7jt (hemat Rp 850rb).

A7CII atau FX30 untuk content creator?

A7CII lebih versatile (foto + video, ringan 514g, IBIS 7-stop). FX30 lebih cinema-grade (4K 120p log, S-Cinetone, design dedicated video). Untuk vlog daily YouTube/TikTok = A7CII. Untuk YouTube channel cinematic + brand sponsorship = FX30. Both sewa Rp 400-525rb/hari.

Apakah perlu lensa zoom atau prime untuk vlog?

Lensa 16-35mm f/2.8 GM jadi pilihan #1 untuk vlog — wide enough untuk handheld self-shooting + ada zoom range untuk medium shot. 50mm prime sebagai second untuk tight shot interview / b-roll cinematic.

Ada paket khusus untuk creator yang baru mulai?

Paket Starter Vlog Rp 500rb/hari — Sony A6400 + 16-50mm + Saramonic Blink 500. Cukup untuk 90% content YouTube/TikTok pemula. Naik level ke Vlogger Setup (Rp 850rb) saat kamu sudah serius build channel.

Bisa long-term rental untuk content creator?

Bisa. Untuk creator yang produce content weekly, sewa 1 minggu+ ada special rate (~25-30% off harga harian). Banyak creator Bandung repeat 2-3x/bulan untuk batch shoot. Chat admin untuk diskusi long-term arrangement.
