Daftar Review

💡 Cara copy review dari Google Maps

  1. Buka profile 7summits Camera di Google Maps
  2. Klik tab Reviews
  3. Copy: nama reviewer, jumlah bintang, tanggal, isi review
  4. Klik + Tambah Review di atas → paste data → Simpan Perubahan → klik Publish

⚠️ Saya gak bisa fetch review dari Google otomatis (mereka block scraping). Manual paste adalah satu-satunya cara legal. Untuk auto-sync butuh Google Places API ($) + Edge Function — bisa di-add later kalau perlu.

Deploy & Publish

Public web di-prerender saat Vercel build. Setelah save di admin, klik Publish untuk trigger rebuild — perubahan muncul ±2-3 menit kemudian.

Status
Memuat...

Setup Deploy Hook

  1. Buka Vercel Dashboard → project sewakamerabandung-website
  2. Settings → Git → scroll ke Deploy Hooks
  3. Create hook: nama admin-publish, branch main → Create Hook
  4. Copy URL hook (format: https://api.vercel.com/v1/integrations/deploy/prj_.../...)
  5. Paste di field bawah ini → Simpan
Disimpan terenkripsi di table admin_config dengan RLS — cuma superadmin yang bisa akses

💡 Tips

  • Publish butuh 2-3 menit (Vercel build + deploy)
  • Tidak perlu publish setiap simpan kecil — kumpulin perubahan dulu, baru publish sekali
  • Branding (warna, font, logo) update real-time tanpa publish — diapply via JS di every page
  • Konten yang butuh publish: lokasi, FAQ, promo, produk enrichment, hero text, paket
